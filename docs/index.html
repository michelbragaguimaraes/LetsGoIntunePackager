Quick Start

# Interactive mode - launches TUI
./letsgointunepackager

# Quiet mode - for CI/CD and automation
./letsgointunepackager -c /path/to/source -s setup.msi -o /path/to/output -q
# Example: Package 7-Zip installer
./letsgointunepackager -c /apps/7zip -s 7z2401-x64.msi -o /output -q

# Output: /output/7z2401-x64.intunewin
